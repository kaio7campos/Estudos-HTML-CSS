Use esse elemento para definir um link
        O link serve para ir a outro documento.

Atributo "href"

Use esse atributo para definir onde quer ir (URL de destino)

Atributo "target"

Use target para definir onde abrir o endereço do link

Temos varios tipos de target

Abre o documento na mesma janela ou guia em que foi clicado

Abre o documento em uma nova janela ou guia

Abre o documento no quadro pai

Abre o documento no corpo inteiro da janela

tipos de URl

URL absoluta tem o endereço da web completo com "https://www"

URL relativa não tem a parte "https://www"

tag serve para lincar qualquer elemento HTML

Coloque os elementos dentro das tags para transformar em link

Use o atributo href para facilitar o acesso
